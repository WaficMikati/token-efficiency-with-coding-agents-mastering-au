# 06.0 Keep Coding Smarter, Not Harder

You've reached the end of the course — and you've covered more ground than it might seem.

You started by understanding what tokens actually are: the units AI models use to process and generate text, and the direct source of cost and latency in every interaction you have with a coding agent. From there you learned that efficiency isn't about typing less — it's about being deliberate: clear prompts, the right context, and the right model for each job.

That last point is what Auto mode is really about. Not ghost text. Not inline suggestions. A routing decision — one that tools like Cursor make automatically and that you can make manually in Copilot — that ensures you're not spending heavily-capable model tokens on tasks that a lighter model handles just as well.

These are small habits individually. Across a real project, they compound. A month of deliberate model selection and concise prompting can meaningfully reduce your API costs and keep your coding sessions faster and more focused.

The best developers using AI tools today aren't the ones who let the agent do everything — they're the ones who know when to let it run and when to steer. You now have the foundation to be one of them.

Go build something.
